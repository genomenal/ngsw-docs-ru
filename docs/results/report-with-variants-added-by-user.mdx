# Отчёт с вариантами, добавленными пользователем

Отчёт может включать только SNVs/Indels, которые были добавлены в него пользователем.
Способ добавления вариантов в отчёт зависит от выбранного типа шаблона:
["Выбранные пользователем SNVs/Indels для добавления в отчёт"](/results/report-with-variants-added-by-user#added-to-report)
или ["Закрепленные SNVs/Indels"](/results/report-with-variants-added-by-user#pinned).

## Выбранные пользователем SNVs/Indels для добавления в отчёт {#added-to-report}

Отчёт основан на блоке
шаблона ["Выбранные пользователем SNVs/Indels для добавления в отчёт"](/settings/report-templates/report-templates#added-to-report).
В блоке выбирается тип мутаций (герминальные или соматические), которые можно добавлять в отчёт.
Если в образце были выявлены варианты с соответствующим типом мутации, отчёт будет содержать
сообщение "*Вы можете отметить варианты в SNV Viewer, чтобы добавить их в этот раздел*".
Включение необходимых вариантов с соответствующим типом мутации в отчёт
описано [здесь](/results/main/snvs-indels/variant-viewer#include-to-report).
Представление SNVs/Indels, включённых в отчёт, описано [ниже](/results/report-with-variants-added-by-user#variants).
Если интерпретация образца была [завершена](/results/sample-interpretation#complete), но
ни один вариант не был включён в отчёт, то отчёт будет содержать
сообщение "*Интерпретация завершена, варианты не добавлены в отчёт*". Тем не менее варианты можно
включить в отчёт и после завершения интерпретации.

#### Исключение вариантов из отчёта {#delete-variant}

Вариант можно исключить из отчёта, наведя курсор мыши не его строку и нажав
на появившуюся корзину <img src={require('/img/results/delete_variant_from_report.png').default} width="23"/>.
В появившемся окне необходимо подтвердить исключение варианта:

<p align="center">
<img src={require('/img/results/delete_variant_confirmation.png').default} width="400"/>
</p>

Такой вариант перестанет быть включённым в отчёт (активированная
кнопка <img src={require('/img/vv/remove_from_report.png').default} width="25"/>) и на странице SNV Viewer.

## Закрепленные SNVs/Indels {#pinned}

Отчёт основан
на [блоке шаблона "Закрепленные SNVs/Indels"](/settings/report-templates/report-templates#pinned-variants).
В блоке выбирается тип мутаций (герминальные или соматические), которые можно добавлять в отчёт.
Если в образце были выявлены варианты с соответствующим типом мутации, отчёт будет содержать
сообщение "*Вы можете закрепить варианты в SNV Viewer, чтобы добавить их в этот раздел*".
Добавление вариантов в отчёт происходит
путём [закрепления их в таблице SNV Viewer](/results/main/snvs-indels/variant-viewer#pin-variant).
Для исключения варианта из отчёта необходимо открепить его на странице SNV Viewer.
Представление SNVs/Indels, добавленных в отчёт, описано [ниже](/results/report-with-variants-added-by-user#variants).
Если интерпретация образца была [завершена](/results/sample-interpretation#complete), но
ни один вариант не был добавлен в отчёт, то отчёт будет содержать
сообщение "*Интерпретация завершена, варианты не добавлены в отчёт*". Тем не менее варианты можно
добавить в отчёт и после завершения интерпретации.

## Представление вариантов в отчёте {#variants}

SNVs/Indels, добавленные в отчёт, представлены в виде таблицы со следующими колонками:

<p align="center">
<img src={require('/img/results/variant_in_report.png').default} width="900"/>
</p>

1. **Ген** - название гена, в котором располагается вариант. При нажатии на ген появится окно со всеми
транскриптами, известными для гена:

<p align="center">
<img src={require('/img/vv/transcripts.png').default} width="600"/>
</p>

Описание колонок таблицы с транскриптами можно найти в описании
[раздела "Транскрипты" на панели детальной информации](/results/main/snvs-indels/snv-details-panel#transcripts).

2. **Позиция** - координаты варианта в геноме (хромосома + начальная позиция).
3. **Генетический вариант** - нуклеотидная и аминокислотная замены
в [номенклатуре HGVS](https://varnomen.hgvs.org/). Нуклеотидная: префикс “c.” (coding; для замены в
кодирующей последовательности) или “n.” (non-coding; для замены в некодирующей последовательности) + геномная
позиция замещенного нуклеотида + референсный аллель > альтернативный аллель. Аминокислотная: префикс “p.” (protein) +
референсная аминокислота + позиция аминокислоты в белке + новая аминокислота, получившаяся в результате замены.
4. **Транскрипт** - идентификатор основного транскрипта гена из базы [RefSeq](https://www.ncbi.nlm.nih.gov/nuccore/)
(NM_xxxxxx.x). При нажатии на транскрипт появится такое же окно со всеми транскриптами, известными для гена, как и
при нажатии на поле "Ген".
5. **Эффект варианта** - последствия варианта для генов. Подробное описание возможных значений
можно посмотреть [здесь](/results/main/snvs-indels/variant-viewer#variant-consequences).
6. **Содержание аллеля** - частота альтернативного аллеля для данного образца (в процентах).
7. **Глубина прочтения** - общее количество прочтений последовательности, перекрывающих позицию варианта, для
данного образца.
8. **Внешние ссылки** - ссылки на вариант в базах данных [dbSNP](https://www.ncbi.nlm.nih.gov/snp/),
[ClinVar](https://preview.ncbi.nlm.nih.gov/clinvar/) и [COSMIC](https://cancer.sanger.ac.uk/cosmic/) (если база
была [загружена в виде пользовательской аннотации](/settings/custom-annotations#define-annotation-type)).
9. Ссылки на вариант во встроенных модулях: <img src={require('/img/vv/igv.png').default} width="30"/> -
модуль визуализации варианта в геноме, <img src={require('/img/vv/open_vv.png').default} width="20"/> -
страница детальной информации о варианте в SNV Viewer
([вкладка "Аннотация"](/results/main/snvs-indels/snv-details-page#annotation-tab)).

Под строкой варианта может быть приведён текст интерпретации варианта, если она была добавлена,
как описано [здесь](/results/main/snvs-indels/variant-interpretation#add-and-edit).

## Экспорт отчётов {#export}

Отчёты с вариантами, добавленными пользователем, можно скачать в формате PDF.
Для этого нажмите на кнопку <img src={require('/img/results/export_pdf.png').default} width="150"/> в
правом верхем углу страницы отчёта.
